~~~
 https://mp.weixin.qq.com/s/67uf7pRxXh7Iwm7MMpqJoA
 BF算法  Brute Force（暴力算法）
 循环比较两个字符串的值,
        先比较第一位
            相同：依次比较后面的
          不相同：增加位数



 RK算法
 BF算法只是简单粗暴地对两个字符串的所有字符依次比较,而RK算法比较的是两个字符串的[哈希值]
